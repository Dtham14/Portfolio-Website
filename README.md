# Daniel Tham Portfolio

A modern, professional portfolio website showcasing my work as a Software Developer and STEM Educator.

## Features

- **Modern Dark Theme** - Refined dark design with amber accents
- **Responsive Design** - Fully mobile-friendly
- **Projects Showcase** - Display of my work including QuizNotes, AI tools, and more
- **Services** - Teaching services for coding, AI, math, music, and 3D modeling
- **Blog** - Articles on AI, education, and technology
- **Contact Form** - EmailJS integration for direct messaging

## Tech Stack

- [Next.js 15](https://nextjs.org/) - React framework
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [React Icons](https://react-icons.github.io/react-icons/) - Icon library
- [EmailJS](https://www.emailjs.com/) - Email functionality

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
├── components/       # React components
│   └── Layout.js     # Main layout with nav and footer
├── data/             # Data files
│   ├── blogposts.js  # Blog content
│   └── services.js   # Services data
├── pages/            # Next.js pages
│   ├── index.js      # Home page
│   ├── projects.js   # Projects showcase
│   ├── contact.js    # Contact form
│   ├── blog/         # Blog pages
│   └── services/     # Services pages
├── public/           # Static assets
└── styles/           # Global styles
```

## Deployment

Deploy easily with [Vercel](https://vercel.com):

```bash
npm run build
```
